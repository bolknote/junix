dialog [<prompt>] [<command>] - переключение в диалоговый режим. Выводиться подсказка (prompt), после ввода строки запускается команда (command) с подменой слова PARM введеной строкой и LEN - длиной введеной строки. После чего эмулятор переходит в режим ввода команды.

Замечание: если нужно использовать переменную в диалоге символ $ нужно заэкранировать слешем (\), апострофы тут не помогут. Напр. dialog [] [echo \$HOME: PARM]

Примеры:
(реализация команды brow)
if ($#) #JS: OpenPage("$1") else dialog [Enter URL: ] [brow PARM]

(реализация команды bc)
if ($#) #JS: $* else dialog [] [if (LEN) {{echo;#JS: PARM;echo >------->}>stdout;bc}]
