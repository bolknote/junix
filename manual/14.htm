if (<выражение>) <команда> - если значение <выражение> "истина", то выполняется команда (можно подставить несколько, заключенных в фигурные скобки) с ее аргументами.

Пример:
if ($?path) echo $path;

if (<выражение>) then;<команда1>;[<команда2>;...][;else; <команда3>;[<команда4>;...];];endif; - другая форма команды. отличается наличием необязательного блока else (команды в нем исполняются, если выражение не верно) и тем фактом, что в блоках можно использовать несколько команд не заключая их в фигурные скобки.

Пример:
if ($?path) then
bc 2+2;echo $path
else
echo None
endif
