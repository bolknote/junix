uname [-s] [-n] [-r] [-v] [-m] [-a] - показывает различную информацию.
-m - тип машины.
-n - имя машины в сети (всегда показывает localhost).
-r - тип операционной системы (версия).
-s - наименование системы.
-v - время и дата создания билда системы.
-a - вся вышеперечисленная информация.

Примеры.
uname -a
uname -v -m
