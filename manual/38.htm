[name]daemon[name]& - это не команда, а модификатор команды. Любая программа (команда), содержащая в своем имени или параметрах подстроку daemon, в фоне (см. &) будет выполняться особым образом - она будет вызываться каждые (примерно) 12 секунд (т.е. работать постоянно). Такой командой в JUnix v5.09b является navidaemon. Приведу пример программы, которая в фоне пытается привести PI к натуральной дроби:

pidaemon:
if ($?pi1) then
if ($pi gt Math.PI) then
set pi2=`bc $pi2+1`
else
set pi1=`bc $pi1+1`
endif
set pi=`bc $pi1/$pi2`
else
set pi1=3 pi2=1 pi=3
endif
Примеры.
maildaemon&;
pidaemon&;
somedaemonthing&;
