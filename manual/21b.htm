mount [[-r] <спецфайл> <путь>] - монтирует устройство. Без параметрой выдает список смонтированных устройств. Все устройства, которые могут быть смонтированы хранятся в каталоге /dev/. Там находятся: navigator - объект браузера navigator (остался по историческим причинам), cookie - cookie вашего браузера, естественно в браузере должна быть включена поддержка cookie (если вы ее не выключали скорее всего так оно и есть), local - если JUNIX у вас на винте или дискете, то это - путь, где он находится, т.е. подмонтировав это устройство вы можете читать из эмулятора исходники JUNIX. В папке local/ находятся флопы и харды, которые можно смонтировать. Для NC этот список состоит из одного элемента и совпадает с спецфайлом local (такова система безопасности), для IE - просто список всех девайсов от A до Z (проверки на сущестовование нет, для ускорения работы). Теперь o грусном при работе с floppy/hard:
1. B NC должна быть включена поддержка Java.
2. Стереть файл под NC не получится. Вызов delete перекрыт зарезервированным словом.
3. В IE все работает только под ActiveX, т.е. Виндами.
4. С Сети это все работать не будет.

Примеры.
mount /dev/navigator /nav
