#<текст> - комментарий ( распространяется до точки с запятой ).
В следующем примере команда echo не выполнится, а clear - выполнится.

Пример.
#echo Плюшка;clear;

#JS: <команда> - выполняет команду JavaScript с эхом в стандартный вывод. Пробел после двоеточия обязателен! Возвращает результат выполнения (в первом примере undefined).

Пример.
#JS: alert("Оооооо!");
#JS: $1

#VB: <команда> - выполняет команду VBScript с эхом в стандартный вывод. Пробел после двоеточия обязателен! Возвращает результат выполнения. Только для IE с установленным VBScript!
